Project Type of assignment, usually carefully planned and researched to reach a specific goal.
Unit Test Small piece of isolated code tested verify functionality, EXTREMELY important.
Backlog An accumulation of tasks unperformed or materials not processed.
Waterfall Process where you have phases that flow like a waterfall, finish one step before moving to the next.
Gantt Chart Project work over a period of time, chart represents data.
Software Requirements Description of system functions/services the system provide.
Task List A task list is a set of tasks that each render on a separate line with a clickable checkbox.
Milestone A certain point of a projects life that is used to measure the progress of the project and to show how close you are to your unlimited goal.
Sprint Focused mini project finished in a set amount of time, 1-4 weeks avg.
Plan Decide on and arrange a task or action in advance.
