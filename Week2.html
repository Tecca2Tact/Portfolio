Vocabulary Webpage (Pt.4)

Article Tag Represents a section of content that forms an independent part of a document.
Optgroup Tag An optgroup is used to group related options in a select element (dropdown)
Video Tag The video tag is used to embed videos directly into a webpage without an external player.
DD Tag The dd tag is used for describing things.
Figure Tag It specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.
Section Tag Defines the section of documents such as chapters, headers, and footers.
DL Tag Defines a description list.
DT Tag Defines term/name in a description list.
Audio Tag Used to embed sound content in documents.
