Vocabulary Page

Zacariah Sutphen

Layer A layer is a group of classes that have the same set of link-time module dependencies to other modules.
Error-Handling The process of responding to and recovering from error conditions in your program.
Cross-Cutting Concerns A concern in code that effects multiple layers at once.
Authentication Verifying the identity of a user, process, or device, often as a prerequisite to allowing access to resources in an information system.
Persistence Layer A software system that is similar to a database and saves/gets data from a storage point.
Error-Messaging The information displayed by the system when an unforeseen problem occurs, usually on a computer or some other devices.
Logging Logging is the act of keeping a log of events that occur in a computer system.
Logic Layer The part of a program when classes determine the information needed to solve problems, manipulate information, and return results to the UI for formatting.
Authorization Allowing access to data/information in a computer system.
UI The UI (User Interface) is the way in which the information on a computer, phone, etc. and instructions on how to use it are arranged on the screen and shown to the user.
Application Software A program designed to handle specific tasks for the user.
Software A set of instructions, data, or programs used to operate computers and execute specific tasks.
